# Scenarios

My GNS3 lab log. Each entry documents a single concept or build I worked through in GNS3 — the goal, the configuration that actually mattered, what broke, and what I took away. New scenarios get added as I go, so this section grows over time instead of needing constant upkeep.

> The physical core (switching, EtherChannel, 802.1X, L3 gateways) is stable infrastructure and is described in the main [README](../README.md#physical-core-on-prem-rack) rather than tracked here. Scenarios cover the GNS3 side, where the active labbing happens.
